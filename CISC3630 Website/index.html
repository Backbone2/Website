But I can't read music…

It's all good. You only need to be able to identify the letter names of the notes on a piano keyboard! This is known as the Chromatic Scale. If you're just starting out (or need a quick reminder) click here. You can also navigate from the link in the Main Menu.

Okay. So what do I need?

 

1. A piano, or a keyboard controller

Let's face it, you can't make music without an instrument! You'll be using this to build your chords and progressions.

Here's a free online Keyboard you can play yourself. This is a great tool that can help you familiarize with the different pitches

 

2. A Pad of paper and a writing utencil.

You're going to want to write down your chords so that way you can practice and build off them.
